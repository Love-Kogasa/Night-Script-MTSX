# Night Script-MTSX 
## Description 简介

*Night MT管理器 JavaScript系语言代码高亮*  
本插件是为手机MT管理器*Javascript开发者*，*Javascript爱好者*编写的一套开源，美观，想对完整，实用，便于开发者阅读/编写代码的语法高亮
